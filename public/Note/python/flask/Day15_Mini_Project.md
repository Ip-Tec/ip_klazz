# Day 15: Mini Project: Simple Todo App

Today you will build a functional Todo App from scratch using everything we’ve learned!

## Requirements:

1. **Model**: A `Todo` class with `id`, `content`, and `date_created`.
2. **Templates**:
   - A homepage showing all todos.
   - A form to add a new todo.
3. **Actions**:
   - Add a todo.
   - Delete a todo via a button.
4. **Styling**: Use a CSS file to make it look clean.

## Tip:

Use `redirect(url_for('index'))` after adding or deleting a todo to refresh the page.

---

## Homework:

Build the Todo App and test the full flow: Add → View → Delete.

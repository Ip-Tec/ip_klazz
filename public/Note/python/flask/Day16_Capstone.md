# Day 16: Capstone Project: Multi-user Blog

Congratulations on making it to the final day! Your capstone project is to build a full **Multi-user Blog**.

## Project Features:

1. **User Accounts**: Registration, Login, and Logout.
2. **Profile System**: Users can update their username and email.
3. **Blog Posts**:
   - Users can Create, Update, and Delete their own posts.
   - Others can only View posts.
4. **Structure**: Organize your project using Blueprints and a `static` folder for styling.

## Final Review

You’ve learned about:

- Routing & Templating
- Database Integration (CRUD & Migrations)
- Security (Hashing & Login)
- Project Organization

What is ARIA?

ARIA (Accessible Rich Internet Applications) is a set of attributes you can add to HTML elements to make web content more accessible to people with disabilities.

Common ARIA Roles

Aria role is used to define the type of an element.

  • role="button" - Indicates an element is a button.
  • role="navigation" - Indicates a navigation section.
  • role="main" - Indicates the main content area.

Common ARIA Attributes

ARIA attributes provide additional information about elements.

  • aria-label - Defines a string that labels the current element.
  • aria-labelledby - Identifies the element (or elements) that labels the current element.
  • aria-describedby - Identifies the element (or elements) that describes the current element.
